# Select

With a select box, users can select one item from a list of values. The selected
item is visible when the select box is closed.

## Installation

To use the input component import the component in your JavaScript.

```typescript
import '@clr/core/select/register.js';
```

```html
<cds-select>
  <label>search</label>
  <select>
    <option>Option One</option>
    <option>Option Two</option>
    <option>Option Three</option>
  </select>
  <cds-control-message>message text</cds-control-message>
</cds-select>
```
