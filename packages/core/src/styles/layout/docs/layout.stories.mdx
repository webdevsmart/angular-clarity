# Layout

Clarity Core components do not provide surrounding white space. Spacing
between components is managed entirely by the `cds-layout` attribute. This
ensures that the system is as composable as possible while keeping white space
predictable and consistent.

The `cds-layout` attribute includes three layout types. Each is designed to
handle particular layout senario.

<Story id="foundation-layout-stories--layout-types" />

## Installation

To use the layout system import the global CSS.

```scss
// import into sass or css
@import '~@clr/core/global.min';

// or import in html
<link href="/node_modules/@clr/core/global.min.css" rel="stylesheet">
```

## Horizontal

The horizontal layout type enables horizontal rows of items that can wrap automatically.
Spacing, alignments and gap between items can all be configured with the component.
[Read More: Horizontal Layout](?path=/docs/experimental-layout-horizontal--page)

<Story id="foundation-layout-stories--horizontal-layout" />

```html
<div cds-layout="horizontal gap:md">
  <cds-placeholder>1</cds-placeholder>
  <cds-placeholder>2</cds-placeholder>
  <cds-placeholder>3</cds-placeholder>
</div>
```

## Vertical

The vertical layout type enables stacks of items. This is useful for spacing
cards, content and other block level items. Spacing, alignments and gap between
stacked items can all be configured with the component.
[Read More: Vertical Layout](?path=/docs/experimental-layout-vertical--page)

<Story id="foundation-layout-stories--vertical-layout" />

```html
<div cds-layout="vertical gap:md">
  <cds-placeholder>1</cds-placeholder>
  <cds-placeholder>2</cds-placeholder>
  <cds-placeholder>3</cds-placeholder>
</div>
```

## Grid

The responsive grid layout type enables columns to be defined for layout. This is useful
for controlling the spacing of items within a container as well as responsive
layout. Spacing, alignments and gap between items can all be configured with
the component. [Read More: Grid Layout](?path=/docs/experimental-layout-grid--page)

<Story id="foundation-layout-stories--grid-layout-columns-responsive" />

```html
<div cds-layout="grid cols@sm:6 cols@md:3 gap:md">
  <cds-placeholder>1</cds-placeholder>
  <cds-placeholder>2</cds-placeholder>
  <cds-placeholder>3</cds-placeholder>
  <cds-placeholder>4</cds-placeholder>
</div>
```

## Responsive

The layout system provides many built-in responsive utilities to make it easy to
manage layouts on many different screen sizes. All responsive utilities follow the same set of breakpoints.

<Story id="foundation-design-tokens-stories--layout" />

```scss
@media (min-width: 576px) {
  // @xs
}

@media (min-width: 768px) {
  // @sm
}

@media (min-width: 992px) {
  // @md
}

@media (min-width: 1200px) {
  // @lg
}

@media (min-width: 1440px) {
  // @xl
}
```
