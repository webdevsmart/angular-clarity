# Toggle

<cds-alert-group status="warning" cds-layout="m-b:lg">

Toggles are used for one to many selection options. Toggles are not for
mutually exclusive lists. Each toggle can be selected independent of the
state of another.

## Installation

To use the toggle component import the component in your JavaScript.

```typescript
import '@clr/core/toggle/register.js';
```

```html
<cds-toggle>
  <label>toggle</label>
  <input type="checkbox" />
  <cds-control-message>message text</cds-control-message>
</cds-toggle>
```
