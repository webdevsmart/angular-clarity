# Date

<cds-alert-group status="danger" cds-layout="m-b:lg">

The date input is a text input with a provided default date icon. The date
input also uses `type="date"` to improve user experience on specific devices
and improve accessibility.

## Installation

To use the date component import the component in your JavaScript.

```typescript
import '@clr/core/date/register.js';
```

```html
<cds-date>
  <label>date</label>
  <input type="date" value="2018-07-22" min="2018-01-01" max="2018-12-31" />
  <cds-control-message>message text</cds-control-message>
</cds-date>
```
