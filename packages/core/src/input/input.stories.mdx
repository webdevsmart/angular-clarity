Text inputs are for relatively short content, such as a single line of text. For
longer input content, such as paragraphs, consider using a textarea.

## Installation

To use the input component import the component in your JavaScript.

```typescript
import '@clr/core/input/register.js';
```

```html
<cds-input>
  <label>text input</label>
  <input type="text" />
  <cds-control-message>message text</cds-control-message>
</cds-input>
```

## Control Width

By default `cds-input` will fill its parent container. Inputs can use the compact
width `control-width="shrink"` which will set the width to the default browser width.
You can apply custom widths by setting a max width on the component.
