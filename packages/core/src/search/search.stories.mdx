# Search

The search input is a text input with a provided default search icon. The search
input also uses `type="search"` to improve user experience on specific devices
and improve accessibility.

## Installation

To use the search component import the component in your JavaScript.

```typescript
import '@clr/core/search/register.js';
```

```html
<cds-search>
  <label>search</label>
  <input type="search" />
  <cds-control-message>message text</cds-control-message>
</cds-search>
```
